# Task 4 - Responsive Website

## Objective
Converted a desktop-only website into a mobile-friendly layout using CSS media queries.

## What I Did
- Added viewport meta tag
- Used flexbox for layout
- Added media query for max-width 768px
- Made navigation vertical on mobile
- Adjusted font size and box width

## Tools Used
- HTML
- CSS
- VS Code
- Chrome DevTools

## Result
Website now works properly on mobile screen sizes.
